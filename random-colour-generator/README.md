# Random Colour Generator
https://daniel-d-dev.github.io/colt-steele-web-dev-course/random-colour-generator/

This is a simple web application that allows users to generate random RGB colours with a click of a button. The background colour of the page will change to a randomly generated colour each time the user clicks the button, and the RGB colour value will be displayed on the screen.

# Technologies Used
* HTML: Used to structure the page and display the elements like the button and header.
* CSS: Used to style the page, including animations and hover effects on the button.
* JavaScript: Handles the functionality of generating random colours and updating the page dynamically.
